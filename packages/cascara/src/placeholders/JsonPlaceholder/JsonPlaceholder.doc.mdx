# JsonPlaceholder

> _NOTE: Placeholder components are only intended for use during development and should not be used in production._

This component can be used during development to display JSON data.

```jsx
---
title: Default display
---
<JsonPlaceholder data={{
  stuff: {
    okay: true,
    something: 23,
    string: 'cool',
  },
}} />
```

```jsx
---
title: Can be open on mount
---
<JsonPlaceholder data={{
  stuff: {
    okay: true,
    something: 23,
    string: 'cool',
  },
}} isInitialOpen />
```

```jsx
---
title: Can display a unique title
---
<JsonPlaceholder data={{
  stuff: {
    okay: true,
    something: 23,
    string: 'cool',
  },
}} title='Custom Title' />
```

```jsx
---
title: If needed, can be customized
---
<JsonPlaceholder
  data={{
  stuff: {
    okay: true,
    something: 23,
    string: 'cool',
  },
}}
  style={{ maxWidth: '50%', backgroundColor: 'red' }}
  title='Style'
/>
```

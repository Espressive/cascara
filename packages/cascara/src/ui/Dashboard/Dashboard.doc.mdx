Dashboards accept an array of config objects that can be used to display widgets.

```jsx
<Dashboard
  config={[
    {
      data: [
        {
          label: 'Total Interactions',
          value: '12,535',
        },
        {
          label: 'Unique Users',
          value: '1,205',
        },
      ],
      title: 'Interactions',
      widget: 'stats',
    },
  ]}
/>
```

# Text Input

A text input is an atom level element that must be used within a form component.

## Existing Text Input Types

- Chat bubble (FAQ editor)
- Integer
- Email
- Password/Sensitive
- Phone number
- Width - Fit - Full width - Multi-line (paragraph, text field)
- States - Active - Focus - Disabled - Error - Success - With placeholder
- Required
- Read only
- Resizable

## Existing Text Input Variations

- With control - Delete/Clear inline
- With character count
- With inline component (examples - labels, dropdowns, icons) - prefix - suffix

## Needed/Suggested/Future Text Input Functionality

- Accessibility
- Limiting/setting character count/input allowance
- States - loading/progress

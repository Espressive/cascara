A data module to display **string** values, it renders a input of type **url**.

### Sandbox

Feel free to play around with the following code:

```jsx
<ModuleSandbox
  isEditing={true}
  record={{
    id: 1,
    url: 'https://example.com',
  }}
>
  <DataUrl
    attribute='url'
    isEditable={true}
    label='Url'
    value='https://example.com'
  />
</ModuleSandbox>
```

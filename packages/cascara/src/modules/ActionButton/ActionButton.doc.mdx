This action module renders a simple button, use it for custom actions.

## Using the module

Additional to the props specified in the aside table, you need to pass the `module` prop, with its value set to **edit**.

### Playground

Feel free to play around with the following code:

```jsx
<ModuleSandbox>
  <ActionButton name='myAction' content='Click me' size='small' />
</ModuleSandbox>
```

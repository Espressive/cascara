A data module to display **string** values, it renders a input of type **text**.

### Sandbox

Feel free to play around with the following code:

```jsx
<ModuleSandbox
  isEditing={true}
  record={{
    id: 1,
    firstName: 'John',
    lastName: 'Doe',
  }}
>
  <DataText
    attribute='firstName'
    isEditable={true}
    label='First Name'
    value='John'
  />
</ModuleSandbox>
```

A data module to display **number** values, it renders a input of type **number**.

### Sandbox

Feel free to play around with the following code:

```jsx
<ModuleSandbox
  isEditing={true}
  record={{
    id: 1,
    ki: 45,
  }}
>
  <DataNumber attribute='number' isEditable={true} label='Number' value={45} />
</ModuleSandbox>
```

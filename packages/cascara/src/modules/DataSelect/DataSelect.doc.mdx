A data module to display multiple option values, it renders a select element.

### Sandbox

Feel free to play around with the following code:

```jsx
<ModuleSandbox
  isEditing={true}
  record={{
    id: 1,
    ki: 45,
  }}
>
  <DataSelect
    attribute='color'
    isEditable={true}
    isLabeled={true}
    label='Color'
    options={[
      {
        key: 'blue',
        value: 'blue',
        text: 'Blue',
      },
      {
        key: 'red',
        value: 'red',
        text: 'Red',
      },
      {
        key: 'white',
        value: 'white',
        text: 'White',
      },
    ]}
    value='blue'
  />
</ModuleSandbox>
```

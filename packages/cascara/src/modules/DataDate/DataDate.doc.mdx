A data module to display **date** values, it renders a input of type **date**.

### Sandbox

Feel free to play around with the following code:

```jsx
<ModuleSandbox
  isEditing={true}
  record={{
    id: 1,
    firstName: 'John',
    lastName: 'Doe',
    date: '2021-07-22',
  }}
>
  <DataDate
    attribute='date'
    isEditable={true}
    label='Date'
    value='2021-07-22'
  />
</ModuleSandbox>
```

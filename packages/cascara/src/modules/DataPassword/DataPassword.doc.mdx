A data module to display **mask** values, it renders a input of type **password**.

### Sandbox

Feel free to play around with the following code:

```jsx
<ModuleSandbox
  isEditing={true}
  record={{
    id: 1,
    firstName: 'John',
    lastName: 'Doe',
    password: 'testpassword',
  }}
>
  <DataPassword attribute='password' isEditable={true} label='Password' />
</ModuleSandbox>
```

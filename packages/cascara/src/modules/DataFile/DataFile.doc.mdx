A data module to upload a **file**, it renders a input of type **file**.

### Sandbox

Feel free to play around with the following code:

```jsx
<ModuleSandbox
  isEditing={true}
  record={{
    id: 1,
    firstName: 'John',
    lastName: 'Doe',
  }}
>
  <DataFile
    attribute='firstName'
    isEditable={true}
    label='My file'
    accept='image/png, image/jpeg'
  />
</ModuleSandbox>
```

# Examples

## Title Markup Only

```jsx
<Title title='Only Title Markup' />
```

# With Pre

```jsx
<Title titlePre='preContent' title='With Pre' />
```

# With Post

```jsx
<Title titlePost='postContent' title='With Post' />
```

# With Pre and Post

```jsx
<Title
  titlePost='postContent'
  titlePre='preContent'
  title='With Pre and Post'
/>
```

# With Sub

```jsx
<Title titleSub='Some Sub Content' title='With Sub Title' />
```
